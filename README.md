# Vishal Portfolio - React Version

This is a responsive portfolio website converted from HTML/CSS/JS to React with TypeScript.

## Features
- Fully responsive design for all devices (mobile, tablet, desktop)
- Modern React components
- Bootstrap 5 for styling
- TypeScript for type safety
- Smooth scrolling navigation
- Interactive project filtering
- Contact form with WhatsApp integration

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Project Structure

```
src/
├── components/

│   └── Footer.tsx          # Footer component
```

## Responsive Design

The website is fully responsive with:
- Mobile-first approach
- Bootstrap grid system
- Custom media queries for optimal display on all screen sizes
- Flexible layouts that adapt to different devices
- Touch-friendly navigation and buttons

## Technologies Used

- React 18
- TypeScript
- Bootstrap 5
- Vite (build tool)
- Font Awesome (icons via images)
